GASPOOL

Pangkalan Omni-Tracker & Peleton (Edisi Open Source)

Sports tracker elit, biaya server Rp 0 (gratis). Alternatif sumber terbuka untuk Strava, ditenagai sepenuhnya oleh arsitektur serverless Cloudflare dan Hono.

✅ KEUNGGULAN (PROS)

  • Biaya Rp 0 (Serverless): Ditenagai Cloudflare D1, R2, dan KV. Tanpa biaya server bulanan.
  • Privasi Mutlak: Data rute dan aktivitas adalah milik pribadi Anda. Tanpa penambangan data.
  • Radar Peleton & Radio: Pantau posisi teman secara langsung dan kirim pesan suara ala Walkie-Talkie.
  • PWA Ringan: Cukup "Add to Homescreen" di browser. Tidak perlu aplikasi berat.

⚠️ KETERBATASAN (CONS)

  • Setup Teknis: Membutuhkan pemahaman dasar terminal dan ekosistem Cloudflare.
  • Tergantung OS HP: GPS mengandalkan browser. OS yang agresif mungkin mematikan GPS jika layar lama dikunci.
  • Tanpa Jejaring Sosial: Tidak ada fitur "Timeline Global" untuk kudos massal seperti Strava.

🔥 Fitur Utama (Persenjataan Pangkalan)

  • Omni-Tracker (Smart GPS): Rekam rute dengan Smart Auto-Pause, Voice Coach per KM, dan Blackbox Recovery.
  • Radar Peleton & Radio PTT: Pantau posisi teman secara 'live'. Komunikasi suara cepat antar anggota peleton.
  • Strava Bridge: Sinkronisasi otomatis aktivitas dari Strava melalui OAuth 2.0.
  • GPX Importer: Impor file .gpx. Jarak dan elevasi dihitung otomatis.
  • Omni Heatmap: Visualisasi riwayat rute di atas peta satelit.
  • Studio Flexing: Buat kartu statistik dan Video Sinematik Animasi Rute (seperti Relive).

🏗️ Arsitektur & Teknologi

Gaspool menggunakan ekosistem serverless modern dan integrasi API pihak ketiga untuk performa maksimal:

Komponen / API Fungsi Utama di Gaspool
Hono Web framework ultra-cepat untuk routing dan backend logic.
Cloudflare D1 Pangkalan data relasional (SQLite) untuk profil, aktivitas, dan sesi.
Cloudflare R2 Penyimpanan objek untuk file koordinat JSON dan rekaman audio radio.
Cloudflare KV Sinkronisasi Radar Peleton dan transmisi Radio PTT secara real-time.
Open-Meteo API Melacak data suhu rata-rata secara otomatis berdasarkan lokasi GPS pengguna.
Strava API Menyinkronkan dan menarik data riwayat olahraga lama pengguna dari Strava.
Cloudflare Turnstile Sistem keamanan anti-bot pada gerbang login tanpa Captcha yang mengganggu.
Leaflet.js Mesin utama untuk merender peta interaktif dan navigasi rute.
Esri & CartoDB Penyedia lapisan peta (Tile Layer) satelit dan Dark Mode yang elegan.
html2canvas Mengkonversi elemen UI menjadi gambar untuk fitur "Share Stats".

🛠️ Panduan Pemasangan (Deployment Protocol)

Fase 1: Persiapan Amunisi

git clone https://github.com/jeannesbryan/gaspool.git
cd gaspool
npm install

Fase 2: Infrastruktur Cloudflare

  • Buat database D1 bernama gaspool-db.
  • Buat bucket R2 bernama gaspool-media (aktifkan Public URL & CORS).
  • Buat KV Namespace bernama gaspool-radar.

Fase 3: Merakit Skema Database

CREATE TABLE users (id INTEGER PRIMARY KEY AUTOINCREMENT, email TEXT UNIQUE, password TEXT);
CREATE TABLE rides (id INTEGER PRIMARY KEY AUTOINCREMENT, name TEXT, distance REAL, moving_time INTEGER, average_speed REAL, max_speed REAL, total_elevation_gain REAL, avg_temp REAL, participants TEXT, start_date DATETIME, polyline TEXT, activity_type TEXT, source TEXT);
npx wrangler d1 execute gaspool-db --file=./schema.sql --remote

Fase 4: Mengamankan Kunci Rahasia

Buat file .dev.vars di folder utama:

JWT_SECRET="KunciRahasiaAnda"
TURNSTILE_SITE_KEY="SiteKeyCloudflare"
TURNSTILE_SECRET_KEY="SecretKeyCloudflare"

Fase 5: Modifikasi Kode

  • Ubah YOUR_R2_PUBLIC_URL_HERE di index.ts & api.ts.
  • Ubah YOUR_USERNAME_HERE di api.ts & dashboard.ts untuk profil publik.

Fase 6: Deploy

npm run deploy

🔑 Login Pertama (Magic Auto-Setup)

Tidak ada tombol Daftar (Register). Saat pertama kali Anda mencoba masuk ke markas dan database users masih kosong, Email dan Password yang Anda masukkan akan otomatis didaftarkan sebagai Kapten (Admin). Setelah itu, pendaftaran terkunci.
